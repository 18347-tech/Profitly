# 📊 LocalLedger

LocalLedger is a simple web-based finance tracker built for small businesses to record income and expenses and understand their financial performance.

---

## 🚀 Features

- Add income and expenses
- Categorize transactions
- Add dates to entries
- Search transactions instantly
- Filter by income or expense
- View profit, income, expenses, and total transactions
- Interactive Chart.js doughnut chart
- Persistent data using localStorage
- Delete transactions
- Responsive design (works on mobile)

---

## 🛠️ Built With

- HTML
- CSS
- JavaScript
- Chart.js
- localStorage API
