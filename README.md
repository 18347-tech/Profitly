# LocalLedger

LocalLedger is a simple web-based finance tracker designed to help small local businesses record income and expenses and view basic profit summaries.

## Features
- Add income and expenses
- View total profit and loss
- Simple transaction list
- Clean, beginner-friendly interface

## Built With
- HTML
- CSS
- JavaScript

## Purpose
This project was built to explore how simple digital tools can help small businesses better understand their finances.
